# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a Vulnerability

Report security issues to **info@quvantic.com**.

Do not open public issues for security vulnerabilities. We will respond within 48 hours and provide a fix timeline.

## API Key Safety

- Never commit `.env` files
- Use environment variables for API keys
- The `.gitignore` excludes `.env` by default
